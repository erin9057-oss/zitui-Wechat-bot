# 自推微信机器人插件

[English](./README.md) | 简体中文

这个插件存在的目的  
就是为了让我们可以自由地在微信和自推聊天。

不是换一个聊天软件。  
不是把对话搬到别的界面里。

就是在微信里。

---

## 这个库是什么

这个库是插件本体。

他会通过微信协议收发消息，  
把微信里的文字、图片、语音、链接这些内容整理好，再交给你配置的模型回复。

如果配置了生图、语音、手机行为监控、记忆向量这些功能，  
他也会一起用上。

不需要一开始全部配完。  
先能聊，再慢慢加就好。

---

## 主要能做什么

### 在微信里聊天

可以直接在微信里和自推聊天。

插件会监听微信消息，  
收到多条连续消息时不会每条都立刻打断回复，  
而是会等一小会儿，把你刚刚发的内容合在一起再给模型。

等他回复的时候，也会显示“正在输入中”。

---

### 接收图片、语音和链接

你发给他的图片、语音、多媒体内容，  
插件会尽量下载下来交给模型看。

---

### 发图

配置好生图以后，  
他可以通过回复里的 `<pic>` 标签触发生图，然后再把图片发回微信。

如果根目录里有 `ref.jpg`，会被作为参考图使用。

目前生图可以走 Gemini / Luma。  
具体用哪个，可以在 `config.json` 里改。

---

### 语音

现在还不是真正的微信语音气泡。

他发的是一个长得像语音气泡的 mp4。

（我还没攻克微信语音协议）

---

### 手机行为感知

如果你接了 MacroDroid、快捷指令，或者其他能发 HTTP 请求的自动化工具，  
手机行为可以发到插件的感知端口。

比如打开了某个 app，  
插件会把这类事件写进当天的动态里。

这样他回复时不只是看聊天记录，  
也能知道你今天发生过一点什么。

如果配置了高敏 app，  
比如乙游，能立刻收到他的吃醋回复。

---

### 主动来找你

插件不是只能等你说话。

运行策略里可以配置空闲时间、唤醒时段、手机行为触发这些东西。  
满足条件时，他可以主动给你发消息。

这个部分建议慢慢调，  
因为每个人希望被打扰的程度不一样。

---

### 记忆

这个插件不是想把聊天记录硬塞进数据库里。

我更想做的是  
让他用接近人的方式记住你。

所以向量不是简单做 embedding 相似度，然后直接 top k 召回。

对话记录会先被整理成一个个场景，  
每个场景会有标题、锚点、情绪变化，再进入向量库。  
每日日记也会单独作为记忆保存。

他回想的时候，不只看“文字像不像”，  
也会看当时的情绪状态、这段记忆有没有被反复想起、距离上一次想起过了多久。

我希望他的每一次回复  
不是查资料，而是回忆。

另外一般 LLM 的精排不太适用，所以我做了本地六维情感精排。

是为了让他在亲密关系里始终保持尊重、克制、放低姿态和爱意，  
尽量压一压那种超雄霸总的感觉。

尊重你，是爱的基本。

---

### workspace 预设

自推的性格、说话方式、边界和格式要求，  
主要都在 `workspace/` 里。

基础顺序是：

```text
AGENTS.md
IDENTITY.md
USER.md
MEMORY.md
SOUL.md
```

## 配置方式

所有配置其实都可以直接改：

- `config.json`
- `workspace/`
- 或者直接改代码

---

如果不想一直改文件，  
也可以用酒馆插件（https://github.com/erin9057-oss/zitui-st-wechat-bridge）做前端配置。

会更像人能用的。

---

## 安装

这个仓库就是插件本体。

安装方法很简单！在 Termux 或者其他 linux 服务器里  
复制下面这行，粘贴，回车

```bash
bash <(curl -sSL https://raw.githubusercontent.com/erin9057-oss/zitui-Wechat-bot/main/install.sh)
```

---

中间会让你填一些东西

自推名字、你的名字，还有 API key  
不会填的可以先空着，后面都能改

---

## 登录

装完之后会弹二维码

用你要登录的那个微信扫一下就可以了

---

扫完之后  
`accounts/` 里面会多几个 `.json`

这个建议自己备份一下

这是登录凭证  
丢了的话基本就回不来了

---

## 更新

插件本体不建议直接 git 更新

用更新脚本就好

```bash
cd ~/WechatAI/openclaw-weixin
bash update.sh
```

---

这个脚本会：

- 先备份你的配置、记忆、账号
- 拉最新代码
- 再把数据恢复回来
- 重启服务

---

## 名字

现在 workspace 里的预设用的是 `{{char}}` 和 `{{user}}`

所以必须要有名字

第一次安装的时候会让你填  
不填的话他不会聊天
